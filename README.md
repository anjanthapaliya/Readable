# Udacity React Nanodegree Project 2
> Readable: A Content And Comment App

## Usage

```bash
cd client
npm install
npm start

cd server
npm install
node server.js
```
Running demo can be found at [https://readable-react.herokuapp.com/](https://readable-react.herokuapp.com/)
